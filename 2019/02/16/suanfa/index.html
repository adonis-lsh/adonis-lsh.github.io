两数之和

https://leetcode-cn.com/problems/two-sum/

自己的答案

public int[] twoSum(int[] nums, int target) {
    int[] twoSumIndex = new int[2];
    for (int i = 0; i < nums.length - 2; i++) {
        for (int j = i + 1; j < nums.length; j++) {
            if (nums[i] + nums[j] == target) {
                twoSumIndex[0] = i;
                twoSumIndex[1] = j;
                return twoSumIndex;
            }
        }
    }
    twoSumIndex[0] = nums.length - 2;
    twoSumIndex[1] = nums.length - 1;
    return twoSumIndex;
}

两数想加

对题目没有理解透彻,最终放弃

官方答案:

整体来说这个思想还是比较重要的,就说这个链表头的问题,一开始就就没有想到这个.真的挺有意思,

public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
    ListNode dummyHead = new ListNode(0);
    ListNode p = l1, q = l2, curr = dummyHead;
    int carry = 0;
    while (p != null  q != null) {
        int x = (p != null) ? p.val : 0;
        int y = (q != null) ? q.val : 0;
        int sum = carry + x + y;
        carry = sum / 10;
        curr.next = new ListNode(sum % 10);
        curr = curr.next;
        if (p != null) p = p.next;
        if (q != null) q = q.next;
    }
    if (carry > 0) {
        curr.next = new ListNode(carry);
    }
    return dummyHead.next;
}

无重复字符的最长子串

https://leetcode-cn.com/problems/longest-substring-without-repeating-characters/

自己的答案

感觉自己做的对算法还是了解太少,脑子中思考的问题的时候,首先想到的都是最笨的方法.....
public int lengthOfLongestSubstring(String s) {
    int maxLength = 0;
    StringBuilder temp = new StringBuilder();
    for (int i = 0; i < s.length(); i++) {
        temp.append(s.charAt(i));
        for (int j = i + 1; j < s.length(); j++) {
            if (!temp.toString().contains(String.valueOf(s.charAt(j)))) {
                temp.append(s.charAt(j));
                if (j == s.length()-1) {
                    if (maxLength < temp.length()) {
                        maxLength = temp.length();
                    }
                    return maxLength;
                }
            } else {
                if (maxLength < temp.length()) {
                    maxLength = temp.length();
                }
                temp.delete(0, temp.length());
                break;
            }
        }
    }

    if (maxLength < temp.length()) {
        maxLength = temp.length();
    }

    return maxLength;
}

寻找两个有序数组的中位数

class Solution {
    public double findMedianSortedArrays(int[] A, int[] B) {
        int m = A.length;
        int n = B.length;
        if (m > n) { // to ensure m<=n
            int[] temp = A; A = B; B = temp;
            int tmp = m; m = n; n = tmp;
        }
        int iMin = 0, iMax = m, halfLen = (m + n + 1) / 2;
        while (iMin <= iMax) {
            int i = (iMin + iMax) / 2;
            int j = halfLen - i;
            if (i < iMax && B[j-1] > A[i]){
                iMin = i + 1; // i is too small
            }
            else if (i > iMin && A[i-1] > B[j]) {
                iMax = i - 1; // i is too big
            }
            else { // i is perfect
                int maxLeft = 0;
                if (i == 0) { maxLeft = B[j-1]; }
                else if (j == 0) { maxLeft = A[i-1]; }
                else { maxLeft = Math.max(A[i-1], B[j-1]); }
                if ( (m + n) % 2 == 1 ) { return maxLeft; }

                int minRight = 0;
                if (i == m) { minRight = B[j]; }
                else if (j == n) { minRight = A[i]; }
                else { minRight = Math.min(B[j], A[i]); }

                return (maxLeft + minRight) / 2.0;
            }
        }
        return 0.0;
    }
}
